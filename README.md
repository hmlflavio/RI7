## Como executar este projeto

Este projeto aplica o conceito de Interfaces e segregação de comportamentos em JavaScript puro.

1. Abre o terminal na pasta do projeto.
2. Certifica-te de que o ficheiro `package.json` possui a propriedade `"type": "module"`.
3. Executa o programa com o comando:
   `node index.js`
